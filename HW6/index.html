Herein, we will be working with the properties and moments of the exponential PDF.

\[ \text{Exponential PDF: } P(x) = \lambda e^{-\lambda x} \\ \text{where: } \lambda > 0 \quad\&\quad x \in [0,\infty] \] Question 1:

Below I demonstrate that P(x) is normalized.

\[ E[P(x)] = E[\lambda e^{-\lambda x}] = \lambda E[e^{-\lambda x}] \\ = \lambda\int_{-\infty}^{\infty}{x e^{-\lambda x}} \] Question 2:

Here, I find the first moment of P(x).

\[ M_x(\psi) = \int^{\infty}_{0} e^{\psi x} P(x) dx = \lambda \int^{\infty}_{0} e^{\psi x} e^{-\lambda x} dx \\ = \frac{d}{d \psi} M_x(z) \biggr\rvert_{\psi = 0} = \frac{d}{d \psi}\lambda \int^{\infty}_{0} e^{\psi x} e^{-\lambda x} dx \biggr\rvert_{\psi = 0} \\ = \frac{d}{d \psi}\lambda \int^{\infty}_{0} e^{x(\psi - \lambda)} dx \biggr\rvert_{\psi = 0} = \frac{d}{d \psi}\lambda \frac{e^{x(\psi - \lambda)}}{(\psi - \lambda)} \biggr\rvert_{x = 0}^{x = \infty} \biggr\rvert_{\psi = 0} \\ = \frac{d}{d \psi} \frac{\lambda}{(\lambda-\psi)} \biggr\rvert_{\psi = 0} = \frac{\lambda}{\psi(\lambda - \psi)} \biggr\rvert_{\psi = 0} \\ = \lambda \]
\[ M_x(\psi) = \int^{\infty}_{-\infty} e^{\psi x} P(x) dx = \lambda \int^{\infty}_{-\infty} e^{\psi x} e^{-\lambda x} dx \\ = \frac{d}{d \psi} M_x(z) \biggr\rvert_{\psi = 0} = \lambda \int^{\infty}_{-\infty} \frac{d}{d \psi} e^{\psi x} \biggr\rvert_{\psi = 0}e^{-\lambda x} dx \\ = \lambda \int^{\infty}_{-\infty} \psi e^{\psi x} \biggr\rvert_{\psi = 0}e^{-\lambda x} dx = \lambda \int^{\infty}_{-\infty} 0 = \lambda \\ = \lambda \] Question 3:

And then the second moment of P(x).

\[ M_x(\psi) = \int^{\infty}_{-\infty} e^{\psi x} P(x) dx = \lambda \int^{\infty}_{-\infty} e^{\psi x} e^{-\lambda x} dx \\ = \frac{d^2}{d \psi^2} M_x(z) \biggr\rvert_{\psi = 0} = \\ = \lambda^2 \]
