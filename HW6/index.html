Herein, we will be working with the properties and moments of the exponential PDF.

\[ \text{Exponential PDF: } P(x) = \lambda e^{-\lambda x} \\ \text{where: } \lambda > 0 \quad\&\quad x \in [0,\infty] \] Question 1:

Below I demonstrate that P(x) is normalized, which means that over its domain, the intergral = 1.

\[ \int^{\infty}_0 P(X) dx = \lambda \int^{\infty}_0 e^{-\lambda x} dx = \lambda \frac{1}{-\lambda} e^{-\lambda x} \biggr\rvert^{\infty}_{x = 0} \\ = -(\frac{1}{e^\infty} - \frac{1}{e^0}) = 1 \] Question 2:

Here, I find the first moment of P(x).

\[ M_x(\psi) = \int^{\infty}_{0} e^{\psi x} P(x) dx = \lambda \int^{\infty}_{0} e^{\psi x} e^{-\lambda x} dx \\ \lt x^1> = \frac{d}{d \psi} M_x(z) \biggr\rvert_{\psi = 0} = \lambda \int^{\infty}_{0} e^{-\lambda x} \frac{d}{d \psi} e^{\psi x} \biggr\rvert_{\psi = 0} dx \\ = \lambda \int^{\infty}_{0} x e^{\psi x} \biggr\rvert_{\psi = 0}e^{-\lambda x} dx = \lambda \int^{\infty}_{0} xe^{-\lambda x} dx = \frac{\lambda}{\lambda^2} \\ \lt x^1> = \frac{1}{\lambda} \] Question 3:

And then the second moment of P(x).

\[ \lt x^2> = \frac{d^2}{d \psi^2} M_x(z) \biggr\rvert_{\psi = 0} = \lambda \int^{\infty}_{0} e^{-\lambda x} \frac{d^2}{d \psi^2} e^{\psi x} \biggr\rvert_{\psi = 0} dx \\ = \lambda \int^{\infty}_{0} e^{-\lambda x} \frac{d}{d \psi} xe^{\psi x} \biggr\rvert_{\psi = 0} dx = \lambda \int^{\infty}_{0} x^2 e^{\psi x}\biggr\rvert_{\psi = 0} e^{-\lambda x} dx \\ = \lambda \int^{\infty}_{0} x^2 e^{-\lambda x} dx = \frac{2\lambda}{\lambda^3} \\ \lt x^2> = \frac{2}{\lambda^2} \] Question 4:

Below I calculate the CV of the distribution.

\[ CV = \frac{\sigma}{\mu} = \frac{\sqrt{\lt x^2>}}{\lt x^1>} = \frac{\sqrt{\frac{1}{\lambda^2}}}{\frac{1}{\lambda}} = 1 \] Question 5:

Finally I calculate the CDF of the distribution.

\[ CDF(v) = \int^{v}_{0} P(x) dx = \int^{v}_{0} \lambda e^{-\lambda x} dx \\ = -e^{-\lambda x} \biggr\rvert^{v}_{x=0} = 1 - e^{-\lambda v} \]
