Herein, we will be working with the properties and moments of the exponential PDF.

\[ \text{Exponential PDF: } P(x) = \lambda e^{-\lambda x} \\ \text{where: } \lambda > 0 \quad\&\quad x \in [0,\infty] \] Question 1:

Below I demonstrate that P(x) is normalized, which means that its values range from [0, 1]

\[ \lambda > 0 \;\&\; e^{-\lambda x} > 0 \;\therefore\; P(X) = \lambda e^{-\lambda x} > 0 \\ P(x) = \lambda e^{-\lambda x} < 1 \;\rightarrow\; \lambda < e^{\lambda x} \] Question 2:

Here, I find the first moment of P(x).

\[ M_x(\psi) = \int^{\infty}_{0} e^{\psi x} P(x) dx = \lambda \int^{\infty}_{0} e^{\psi x} e^{-\lambda x} dx \\ = \frac{d}{d \psi} M_x(z) \biggr\rvert_{\psi = 0} = \lambda \int^{\infty}_{0} e^{-\lambda x} \frac{d}{d \psi} e^{\psi x} \biggr\rvert_{\psi = 0} dx \\ = \lambda \int^{\infty}_{0} x e^{\psi x} \biggr\rvert_{\psi = 0}e^{-\lambda x} dx = \lambda \int^{\infty}_{0} xe^{-\lambda x} dx = \frac{\lambda}{\lambda^2} \\ = \frac{1}{\lambda} \] Question 3:

And then the second moment of P(x).

\[ = \frac{d^2}{d \psi^2} M_x(z) \biggr\rvert_{\psi = 0} = \lambda \int^{\infty}_{0} e^{-\lambda x} \frac{d^2}{d \psi^2} e^{\psi x} \biggr\rvert_{\psi = 0} dx \\ = \lambda \int^{\infty}_{0} e^{-\lambda x} \frac{d}{d \psi} xe^{\psi x} \biggr\rvert_{\psi = 0} dx = \lambda \int^{\infty}_{0} x^2 e^{\psi x}\biggr\rvert_{\psi = 0} e^{-\lambda x} dx \\ = \lambda \int^{\infty}_{0} x^2 e^{-\lambda x} dx = \frac{\lambda}{\lambda^3} \\ = \frac{1}{\lambda^2} \] Question 4:

Below I calculate the CV of the distribution.

\[ CV = \frac{\sigma}{\mu} = \frac{\sqrt{}}{} = \frac{\sqrt{\frac{1}{\lambda^2}}}{\frac{1}{\lambda}} = 1 \] Question 5:

Finally I calculate the CDF of the distribution.

\[ CDF(v) = \int^{v}_{0} P(x) dx = \int^{v}_{0} \lambda e^{-\lambda x} dx \\ = -e^{-\lambda x} \biggr\rvert^{v}_{x=0} = 1 - e^{-\lambda v} \]
