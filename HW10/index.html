Part 1

I begin by deriving the PDF for this system from the given parameters:

\[ P(t_{tot}=t) = P(t) = \text{Prob. it takes a time "t" to complete the linear chain of 2 events.} \\ \text{Rates are: } \lambda_1 + \lambda_2 \\ \text{We split our "timeline" into two regions, s is the time for event 1} \\ \text{while t-s is the time for event 2} \\ P(t) = P(t_1=s) \land P(t_2 = t-s) \text{ for all values of s < t; is mutually exclusive} \\ \therefore\; P(t) = \int_{0}^{t} P(t_1=s)P(t_2=t-s) ds \\ = \int_{0}^{t} [\lambda_1 e^{-\lambda_1 s}] [\lambda_2 e^{-\lambda_2 (t-s)}]ds = \lambda_1 \lambda_2 \int_{0}^{t} exp[-\lambda_1(s) - \lambda_2 (t-s)] ds \\ = \lambda_1 \lambda_2 \int_{0}^{t} exp[s(\lambda_2-\lambda_1)]exp[-\lambda_2 t] ds = \lambda_1 \lambda_2 e^{-\lambda_2 t} \int_{0}^{t} e^{s(\lambda_2-\lambda_1)} ds \\ = \frac{\lambda_1 \lambda_2 e^{-\lambda_2 t}}{(\lambda_2-\lambda_1)} [e^{s(\lambda_2-\lambda_1)}]_{s=0}^{s=t} = \frac{\lambda_1 \lambda_2 e^{-\lambda_2 t}}{(\lambda_2-\lambda_1)} [e^{t(\lambda_2-\lambda_1)}-1] \\ P(t) = \frac{\lambda_1 \lambda_2}{(\lambda_2-\lambda_1)} [e^{-\lambda_1t}-e^{-\lambda_2 t}] \]

Part 2

I then derive the mean of the distribution:

\[ = \int_{0}^{\infty} t P(t) dt = \frac{\lambda_1 \lambda_2}{(\lambda_2-\lambda_1)} \int_{0}^{\infty} t[e^{-\lambda_1t}-e^{-\lambda_2 t}] dt \\ = \frac{\lambda_1 \lambda_2}{(\lambda_2-\lambda_1)} [\frac{1}{\lambda_1^2} - \frac{1}{\lambda_2^2}] = \frac{1}{(\lambda_2-\lambda_1)} [\frac{\lambda_2}{\lambda_1} - \frac{\lambda_1}{\lambda_2}] \]

Part 3

I then find the limit of P(t) as the lambdas assume the same value:

\[ \text{let } Z(t) = lim_{\lambda_2 \to \lambda_1} P(t) \\ Z(t) = \frac{\lambda_1 \lambda_2}{(\lambda_2-\lambda_1)} [e^{-\lambda_1t}-e^{-\lambda_2 t}] = \frac{0}{0} \\ \text{By l'hospital's rule: } Z(t) = lim_{\lambda_2 \to \lambda_1} \frac{A}{B} \\ \text{let } A = \frac{d}{d \lambda_2} [\lambda_1 \lambda_2][e^{-\lambda_1t}-e^{-\lambda_2 t}] = \lambda_1 e^{-\lambda_1t} - \frac{d}{d \lambda_2} \lambda_1 \lambda_2 e^{-\lambda_2 t} \\ = \lambda_1 e^{-\lambda_1t} - \lambda_1[- \lambda_2 t e^{-\lambda_2 t} + e^{-\lambda_2 t}] = \lambda_1[e^{-\lambda_1t} + \lambda_2 t e^{-\lambda_2 t} - e^{-\lambda_2 t}] \\ \text{let } B = \frac{d}{d \lambda_2} [\lambda_2-\lambda_1] = 1 \\ \therefore\; Z(t) = lim_{\lambda_2 \to \lambda_1} A = \lambda_1^2 t e^{-\lambda_1 t} \]
