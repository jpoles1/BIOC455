Part A:
\[ P(x) = \mu e^{-\mu x} \;\land\; \mu > 1 \;\land\; x \geq 0 \\ M_x(\lambda) = \lt e^{\lambda x}> = \int_{0}^{\infty} e^{\lambda x} P(x) dx \\ = \mu \int_{0}^{\infty} e^{\lambda x} e^{-\mu x} dx = \mu \int_{0}^{\infty} e^{-x(\mu - \lambda)} dx \\ \text{If we set: } Q = x(\mu - \lambda) \;\land\; dx = \frac{dQ}{\mu - \lambda} \\ M_x(\lambda) = \frac{\mu}{\mu - \lambda} \int_{0}^{\infty} e^{-Q} dQ = \frac{\mu}{\mu - \lambda} = (1 - \frac{\lambda}{\mu})^{-1} \\ \]
Part B:
Using the above moment generating function: \[ \lt x^n> = \frac{d^n}{d\lambda^n} M_x(\lambda) |_{\lambda = 0} \\ \lt x^0> = M_x(\lambda)|_{\lambda = 0} = \frac{\mu}{\mu} = 1 \\ \frac{d}{d\lambda} (1 - \frac{\lambda}{\mu})^{-1} = \frac{1}{\mu}(\frac{\lambda}{\mu}-1)^{-2} \\ \lt x^2> = \frac{d^2}{d\lambda^2} (1 - \frac{\lambda}{\mu})^{-1} |_{\lambda = 0} = -\frac{2}{\mu^2}(\frac{\lambda}{\mu}-1)^{-3} |_{\lambda = 0} = \frac{2}{\mu^2} \\ \frac{d^3}{d\lambda^3} (1 - \frac{\lambda}{\mu})^{-1} = \frac{6}{\mu^3}(\frac{\lambda}{\mu}-1)^{-4} \\ \lt x^4> = \frac{d^4}{d\lambda^4} M_x(\lambda)|_{\lambda = 0} = -\frac{24}{\mu^4}(\frac{\lambda}{\mu}-1)^{-5}|_{\lambda = 0} = \frac{24}{\mu^4} \]
