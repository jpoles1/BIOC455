\[ x_\epsilon = x_s + \epsilon \\ \dot{x}_\epsilon = \frac{d}{dt} (x_s + \epsilon) = \dot{\epsilon} = \beta - \gamma (x_s + \epsilon) + \tilde{\eta}(t) \\ \dot{\epsilon} = \beta - \gamma x_s - \gamma \epsilon + \tilde{\eta}(t) = - \gamma \epsilon + \tilde{\eta}(t) \\ \]

Part 3

I begin by deriving the fixed point of the ODE representation of this system:

\[ \dot{x} = \frac{\alpha}{1 + \frac{x}{c}} - \gamma x = 0 = \alpha - \gamma x (1+ \frac{x}{c}) \\ = \alpha - \gamma x + \frac{\gamma}{c} x^2 \;\therefore\; x_s = \frac{-c \pm \sqrt{c^2 + \frac{4 \alpha c}{\gamma}}}{2} \\ \text{We only consider the case where:} \\ x_s = \frac{-c + \sqrt{c^2 + \frac{4 \alpha c}{\gamma}}}{2} \]

I will then describe the first-order dynamics of perturbations about the above fixed point using a SDE. Unlike the last system, this one is non-linear. As such, a taylor approximation of the perturbation will yield a sufficiently accurate estimate:

\[ A = A_0 + \epsilon A_1 + \epsilon^2 A_2 + ... \\ A \approx A_0 + \epsilon A_1 \;\text{ given } O(\epsilon^1) \\ \therefore\; \dot{x}_\epsilon = \dot{\epsilon} \approx A_0 + A_1 \epsilon \\ = [\frac{\alpha}{1 + \frac{x_s}{c}} - \gamma x_s] + \frac{d}{d\epsilon}[\frac{\alpha}{1 + \frac{x_s + \epsilon}{c}} - \gamma (x_s + \epsilon)] |_{\epsilon=0}\epsilon \\ = 0 + [\frac{-a/c}{(1+x_s/c)^2} - \gamma]\epsilon \]
